01 — Contexto do Desafio

A história por trás do app

O Gourmet Garça é o restaurante mais querido da cidade. Mas há um problema: na hora do rush do almoço, os garçons perdem tempo calculando o total de cada pedido na mão, e os erros são frequentes. O dono, Seu Antônio, pediu ajuda para você — o(a) único(a) desenvolvedor(a) mobile da cidade.

Ele não precisa de nada sofisticado: quer um app onde o garçom seleciona os pratos do pedido e vê o valor total se atualizar na hora, sem botão de confirmar, sem recarregar tela. Simples, rápido e confiável.

03 — Roteiro de execução

Passo a passo para construir o app

Leia todos os passos antes de começar a codar. Entender a estrutura completa antes de escrever a primeira linha evita retrabalho.
1

Crie o projeto no Android Studio

Novo projeto → Empty Activity com Jetpack Compose habilitado. Nomeie o projeto GourmetGarca, package br.com.fatec.gourmetgarca, linguagem Kotlin, API mínima 26.

Android Studio
2

Modele os dados do cardápio

Crie um arquivo Prato.kt com uma data class que represente cada item do menu. Defina pelo menos 6 pratos na lista de cardápio (use o código base como referência).

data class · Kotlin
3

Construa o composable da tela principal

Crie CardapioScreen() como função composable. Use Scaffold com TopAppBar para o cabeçalho "Gourmet Garça". Dentro do body, use Column com LazyColumn para listar os pratos.

Compose · Scaffold
4

Gerencie o estado dos itens selecionados

Declare val selecionados = remember { mutableStateListOf<Prato>() } dentro de CardapioScreen. Este é o coração reativo do app: qualquer mudança nessa lista dispara a recomposição automática.

remember · mutableStateListOf
5

Crie o composable de item do cardápio

Componente PratoItem(prato, selecionado, onToggle) com um Row contendo Checkbox, nome do prato e preço formatado. O toggle adiciona ou remove o prato da lista selecionados.

Checkbox · Row · Modifier
6

Exiba o total calculado em tempo real

Calcule val total = selecionados.sumOf { it.preco } e exiba em um Card fixo no rodapé da tela, formatado como moeda: NumberFormat.getCurrencyInstance(Locale("pt","BR")).

sumOf · NumberFormat
7

Teste e valide o comportamento reativo

Execute no emulador ou dispositivo físico. Marque e desmarque pratos e confirme que o total atualiza instantaneamente, sem nenhum botão de "calcular". Isso comprova que a recomposição está funcionando corretamente.

Emulador · Validação

05 — Avaliação

Rubrica de correção

Critério Descrição Pontos
Data class correta A data class Prato possui os campos necessários com os tipos corretos. A lista cardapioGourmet tem ao menos 6 itens preenchidos. 1,5
Estado reativo funcional Uso correto de remember { mutableStateListOf() }. Ao marcar/desmarcar um item, o total é atualizado imediatamente sem necessidade de botão. 2,5
Layout com Compose Uso correto de Scaffold, TopAppBar, LazyColumn, Row e Modifier. Layout responsivo e sem overflow de conteúdo. 2,0
Componente PratoItem Componente reutilizável com Checkbox, nome e preço do prato. Recebe estado e callback como parâmetros (sem lógica acoplada). 2,0
Formatação de moeda O total é exibido em formato brasileiro (R$ X,XX) usando NumberFormat ou String.format. Não aceita concatenação manual de "R$". 1,0
Qualidade e organização Código em arquivos separados, nomes em português ou inglês consistentes, sem código comentado ou imports desnecessários. Vídeo demonstra o app funcionando. 1,0
Total da atividade 10,0

06 — Para quem quer ir além

Desafios bônus

Os bônus não substituem pontos da rubrica — são para quem quer se destacar e aprofundar o aprendizado.

Bônus Bronze

Botão "Limpar pedido"

Adicione um botão que chama selecionados.clear() e desmarca todos os itens de uma vez. Testa o entendimento de operações em listas reativas.

Bônus Prata

Contador de itens selecionados

Exiba "X item(ns) selecionado(s)" dinamicamente na TopAppBar ou acima do total. Reforce o uso de expressões dentro de composables.

Bônus Prata

Filtro por categoria

Agrupe os pratos por categoria (Entrada, Prato Principal, Sobremesa) com cabeçalhos separadores na LazyColumn usando stickyHeader.

Bônus Ouro

Campo de quantidade por item

Substitua o Checkbox por um seletor +/- de quantidade. Recalcule o total considerando quantidade × preço para cada item.
